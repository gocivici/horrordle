Instructions

×

Use the scene provided to guess the name of the movie, each scene features a character from the movie.

If you guess wrong or skip, a new scene from the movie is revealed. You have a maximum of 3 guesses to find the movie.

you can use the buttons below the image to navigate once they are revealed.

If you win the first round, you'll have the chance to guess which scene features the character that survives the movie in the bonus stage.

Horrordle will only run through Spooktober! make sure to check everyday until November.
